Architecture

Five agents. One truth.

A LangGraph state machine orchestrates four specialised AI agents in sequence — each one building on the last to produce grounded, adversarially-reviewed verdicts.

AGENT 01
⚙
Claim Extractor
LLaMA 3.3 70B with Pydantic structured output. Pulls every distinct factual claim from raw text.
AGENT 02
⬡
Evidence Hunter
HyDE retrieval against 4,210 Guardian + Wikipedia chunks in pgvector. Finds real cited sources.
AGENT 03
◈
Synthesis Agent
Reasons strictly from retrieved evidence. Returns TRUE / FALSE / UNCERTAIN with confidence score.
AGENT 04
⚔
Critic Agent
Adversarial review. Flags overconfidence, unsupported claims, and alternative interpretations.
AGENT 05
✦
Correction Agent
Fires on FALSE / UNCERTAIN verdicts. Searches for what is actually true and surfaces the correct information with sources.
